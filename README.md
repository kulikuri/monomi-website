- Video background hero section with placeholder

## Technical Details
### Typography:
- **Headings**: Bebas Neue, letter-spacing: 0.5px
- **Body**: Source Sans Pro, line-height: 1.6
- **Colors**: #ffffff, #cccccc, #999999 for hierarchy

### Color Palette:
- **Primary**: #0C0D19 (Dark Blue)
- **Secondary**: #ffffff (White)
- **Mobile Sidebar**: #1a1d2e (Darker Blue)
- **Borders**: #2a2a2a, #2a2d3e (Mobile)

### Responsive Breakpoints:
- **Mobile**: ≤768px (Sidebar navigation)
- **Tablet**: ≤1024px (Adjusted text sizes)
- **Desktop**: >1024px (Full layout)

### Animation System:
- **Easing**: cubic-bezier(0.16, 1, 0.3, 1) for professional feel
- **Duration**: 1.2s for elegant, deliberate animations
- **Staggering**: JavaScript-controlled delays (150ms, 300ms, 450ms, 600ms)
- **Hero Timing**: 500ms initial delay + 300ms between elements

### Key Components:
1. **Transparent/Sticky Navbar**: Changes background on scroll
2. **Sidebar Mobile Menu**: 320px sliding menu with overlay
3. **Video Placeholder**: For testing gradient overlays
4. **Service Cards**: Clean design without badges
5. **Smooth Animations**: Professional scroll-triggered reveals
